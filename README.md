# Prehouses.es - Marketplace de Casas Prefabricadas

Este proyecto es un portal web estático diseñado para ser un marketplace de casas prefabricadas en España.

## Características
- Diseño moderno y premium con UI/UX optimizada.
- Color principal: #234DD3.
- Páginas: Inicio, Catálogo, Usuarios.
- Totalmente responsive.

## Despliegue en Render

Para publicar este sitio en Render como un sitio estático:

1. Ve a [dashboard.render.com](https://dashboard.render.com).
2. Haz clic en **New +** y selecciona **Static Site**.
3. Conecta tu repositorio de GitHub: `Gerardmc01/prehouses.es`.
4. Configuración:
   - **Name**: prehouses (o lo que prefieras)
   - **Branch**: main
   - **Root Directory**: (déjalo en blanco)
   - **Build Command**: (déjalo en blanco)
   - **Publish Directory**: . (un punto, o déjalo en blanco si Render lo detecta)
5. Haz clic en **Create Static Site**.

¡Tu web estará online en segundos!
